## 프로퍼티 설정
* H2 설정
```
spring:
  datasource:
  url: jdbc:h2:tcp://localhost/~/jpashop
  username: sa
  password:
  driver-class-name: org.h2.Driver
```

* ddl 옵션
```
jpa:
 hibernate:
 ddl-auto: create
```

* 쿼리 로그
```
format_sql: true
```

* 로그 레벨
```
logging.level:
 org.hibernate.SQL: debug
 org.hibernate.type: trace
```
